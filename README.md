# Titanic Survival Prediction using Logistic Regression

## Objective
Predict whether a passenger survived the Titanic disaster using Logistic Regression.

## Dataset
Seaborn Titanic Dataset

## Steps Performed
- Data Cleaning
- Handling Missing Values
- One Hot Encoding
- Feature Scaling
- Train-Test Split
- Logistic Regression Model Training
- Model Evaluation

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix
- ROC Curve and AUC Score

## Result
The model achieved good classification performance with reliable ROC-AUC score.

## Tools Used
Python, Pandas, Scikit-learn, Matplotlib, Seaborn
